Guía básica de JavaScript

Este documento muestra los principales conceptos para empezar con JS: funciones, arrays, if, while, for y ejemplos con fetch usando APIs reales.

1. Tipos de funciones

En JavaScript hay varias formas de definir funciones. Todas hacen lo mismo, pero se usan en contextos distintos.

1.1 Declaración de función

Este es el tipo más clásico. La función se puede usar antes o después de declararla.

function saludar(nombre) {
    return `Hola, ${nombre}!`;
}

1.2 Expresión de función

La función se guarda en una variable. No se puede usar antes de definirla.

const multiplicar = function(a, b) {
    return a * b;
};

1.3 Función flecha

Es una forma más corta y moderna. Ideal para funciones simples.

const suma = (a, b) => a + b;
const doble = valor => valor * 2;

1.4 Callback

Una función puede recibir otra función como parámetro. Eso se llama callback.

function ejecutarConValor(valor, callback) {
    return callback(valor);
}

const resultado = ejecutarConValor(5, num => num + 10);
console.log(resultado); // 15

2. Arrays

Un array guarda varios valores dentro de una sola variable.

const frutas = ["manzana", "plátano", "kiwi"];
console.log(frutas[0]); // manzana
frutas.push("pera"); // añade al final
frutas.pop(); // quita el último elemento

También podemos recorrer los arrays con diferentes bucles.

2.1 Bucle for

for (let i = 0; i < frutas.length; i++) {
    console.log(i, frutas[i]);
}

2.2 for...of

for (const fruta of frutas) {
    console.log(fruta);
}

2.3 forEach

frutas.forEach((fruta, indice) => {
    console.log(indice, fruta);
});

3. if / else

Se usa cuando queremos tomar decisiones en el código.

const edad = 18;
if (edad >= 18) {
    console.log("Puedes entrar");
} else {
    console.log("Aún no puedes entrar");
}

También existe else if para más opciones.

const temperatura = 30;
if (temperatura > 35) {
    console.log("Hace mucho calor");
} else if (temperatura >= 25) {
    console.log("Hace buen tiempo");
} else {
    console.log("Hace fresco");
}

4. while

El bucle while repite código mientras la condición sea verdadera.

let contador = 0;
while (contador < 5) {
    console.log("contador =", contador);
    contador++;
}

5. for

El bucle for es muy útil cuando conocemos el número de repeticiones.

for (let i = 1; i <= 5; i++) {
    console.log("Paso", i);
}
